## Torch Profiling scripts

* `profile_tcuscan_ops.py`: Ascend/NPU profiling scripts
* `profile_tcuscan_ops_gpu.py`: GPU/CUDA profiling scripts
* `profile_tcuscan_ops_cpu.py`: CPU profiling scripts
* `profile_sparse_matrices.py`: sparse Suite matrices profiling scripts
* `profile_random_matrices.py`: random sparse matrices profiling scripts

Note the GPU/CUDA scripts can be merged into a single file but split into seperate files for convenience.
